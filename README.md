# Litres — Голосовой помощник по истории для 5 класса

Интерактивный веб-помощник для подготовки домашнего задания по **Истории Древнего мира** (5 класс, учебник Кошелева, часть 2). Ученик выбирает параграф, отвечает голосом на вопросы, получает озвученные AI-пояснения с иллюстрациями, картами событий и проходит тесты по датам.

---

## Идея проекта

Голосовой помощник с базой данных учебника, ElevenLabs (TTS) и Gemini (LLM), который:

1. Принимает номер параграфа от ученика
2. Поэтапно задаёт вопросы из учебника (голосом)
3. Ученик отвечает голосом (или текстом)
4. AI поясняет ответ в 5-10 предложениях + интересные факты + сравнение с современностью
5. В конце — тест по датам параграфа
6. Обобщающий тест по разделу (Греция / Рим / Германцы и Славяне)
7. Режим повторения плохо отвеченных вопросов

## Учебник

- **Название:** История Древнего мира, 5 класс (Кошелев, часть 2, 2019)
- **Страниц:** 141, **Параграфов:** 31 (§1–§31)
- **Разделы:**
  - Раздел 1: Древняя Греция (§1–§16)
  - Раздел 2: Древний Рим (§17–§29)
  - Раздел 3: Древние германцы и славяне (§30–§31)

## Flow ученика

```
1. Открывает сайт → главная с картой Древнего мира
2. Вводит никнейм (без пароля) → прогресс сохраняется
3. Выбирает раздел → список параграфов (все 31 доступны)
4. Открывает параграф → иллюстрация + вступление (озвучено)
5. Появляется вопрос №1 + картинка + "Ты знал?" факт
6. Ученик отвечает голосом или текстом
7. Gemini → пояснение 5-10 предложений + факт + ассоциация
8. Пояснение озвучивается (ElevenLabs)
9. Повтор для 3-5 вопросов параграфа
10. Тест по датам (карточки + drag-and-drop таймлайн)
11. Медаль (бронза/серебро/золото)
12. Ошибки → блок "На повторение"
13. Обобщающий тест по разделу
14. Итоги по всем параграфам раздела
```

## Ключевые решения

| Проблема | Решение |
|---|---|
| **Браузеры** | Голос + текстовый fallback, автоопределение |
| **Стоимость ElevenLabs** | Вопросы заранее, пояснения кэшируются (~70% экономии) |
| **Прогресс** | Никнейм + Supabase + LocalStorage |
| **Речь ребёнка** | Умный промпт Gemini — лояльная интерпретация |
| **Безопасность** | Системный промпт: только образовательный контент |

## 9 элементов вовлечения

| Элемент | Описание |
|---|---|
| **"Ты знал?"** | Удивительный факт перед каждым вопросом |
| **Современные аналогии** | "Олимпийские игры = ЧМ по футболу" |
| **Мини-словарь** | Всплывающие подсказки по терминам |
| **Мнемонические карточки** | Яркие ассоциации для дат |
| **Режим повторения** | Только вопросы, где ошибся |
| **Медали** | Бронза/серебро/золото за параграф |
| **Поделиться** | Скриншот-карточка с результатом |
| **Таймлайн** | Drag-and-drop событий на шкалу времени |
| **Карта событий** | Leaflet с метками битв, городов, маршрутов |

### Система подсказок (3 уровня)
1. Лёгкий наводящий вопрос
2. Конкретный намёк с контекстом
3. Полный ответ с пояснением

## Структура страниц

```
/                        → Главная (карта мира, выбор раздела)
/section/[id]            → Раздел (параграфы + прогресс)
/paragraph/[id]          → Параграф (вопросы, карта, иллюстрации)
/paragraph/[id]/quiz     → Тест по датам параграфа
/paragraph/[id]/review   → Повторение ошибок
/section/[id]/quiz       → Обобщающий тест по разделу
/section/[id]/summary    → Итоги по разделу
```

## Supabase схема

```sql
CREATE TABLE students (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  nickname TEXT UNIQUE NOT NULL,
  created_at TIMESTAMPTZ DEFAULT now()
);

CREATE TABLE answers (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  student_id UUID REFERENCES students(id),
  paragraph_id INT NOT NULL,
  question_index INT NOT NULL,
  answer_text TEXT,
  is_correct BOOLEAN,
  hint_level INT DEFAULT 0,
  created_at TIMESTAMPTZ DEFAULT now()
);

CREATE TABLE quiz_results (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  student_id UUID REFERENCES students(id),
  paragraph_id INT,
  section_id TEXT,
  score INT NOT NULL,
  total INT NOT NULL,
  created_at TIMESTAMPTZ DEFAULT now()
);

CREATE TABLE audio_cache (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  text_hash TEXT UNIQUE NOT NULL,
  audio_url TEXT NOT NULL,
  created_at TIMESTAMPTZ DEFAULT now()
);
```
