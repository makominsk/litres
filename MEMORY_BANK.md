# MEMORY BANK — Litres Project

> Актуальный контекст проекта (обновлено: 2026-03-04).

---

## Что это за проект

Интерактивный помощник по истории Древнего мира для 5 класса:
- ученик выбирает раздел и параграф;
- отвечает голосом или текстом;
- получает AI-пояснения;
- проходит тесты по датам и видит прогресс.

---

## Текущее состояние продукта

### Реализовано
- Главная страница с выбором разделов.
- Экран раздела со списком параграфов и прогрессом.
- Экран параграфа с вопросами, подсказками, оценкой ответа.
- Квизы по параграфам и разделам.
- Локальный прогресс через Zustand + persist.
- Интеграция учебника PDF:
  - кнопка открытия на главной;
  - fullscreen viewer с листанием страниц;
  - переход к нужному параграфу (§ -> страница);
  - кнопка открытия учебника сразу на текущем § из страницы параграфа.
- На главной добавлен блок со случайным проверенным историческим фактом с источником.

- Карта переходов параграфов по страницам: `src/data/textbook-pages.json`.
- Страницы привязаны для §1–§31 и сверены по фактическим страницам PDF:
  - §1 начинается на стр. 7
  - §17 начинается на стр. 74

Ключевые файлы интеграции:

## Архитектура и страницы

Маршруты:
- `/` — главная (разделы, учебник, случайный факт)
- `/section/[id]` — параграфы раздела
- `/paragraph/[id]` — вопросы параграфа
- `/paragraph/[id]/quiz` — тест по датам параграфа
- `/paragraph/[id]/review` — повторение ошибок
- `/section/[id]/quiz` — обобщающий тест раздела

API:
- `POST /api/evaluate`
- `POST /api/hint`
- `POST /api/student`
- `POST /api/progress`, `GET /api/progress`
- `POST /api/tts`
- `POST /api/transcribe`

## Важные замечания

- В CI/локальном sandbox `next build` может падать из-за недоступности Google Fonts (сетевое ограничение), даже при корректном коде.
- Для локальной проверки изменений используется `npx tsc --noEmit` как базовая быстрая валидация.
- Глобальный базовый размер шрифта увеличен в `src/app/globals.css`:
  - desktop: `html { font-size: 20px; }`
  - mobile: `html { font-size: 19px; }`
