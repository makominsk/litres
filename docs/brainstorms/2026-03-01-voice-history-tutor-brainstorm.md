# Голосовой помощник по истории для 5 класса

## Что строим

Интерактивный веб-помощник для подготовки домашнего задания по истории Древнего мира (5 класс, учебник Кошелева, часть 2). Ученик выбирает параграф, отвечает голосом на вопросы, получает озвученные пояснения с иллюстрациями и проходит тесты по датам. Без авторизации, mobile-first, работает на любом устройстве.

## Почему этот подход

Выбран подход A (Next.js + Vercel AI SDK) — оптимальный баланс между функциональностью и простотой. Деплой на Vercel, бесплатные тиры Supabase, стриминг ответов.

## Стек технологий

| Компонент | Технология | Примечание |
|---|---|---|
| Frontend | Next.js 14 (App Router) | Mobile-first, адаптивный |
| LLM | Google AI Studio (Gemini) | Основной мозг |
| Картинки | Pre-generated через Google AI Studio | Статические, по параграфам |
| Голос ученика (STT) | Web Speech API + текстовый fallback | Голос в Chrome/Edge, текст везде |
| Голос учителя (TTS) | ElevenLabs (мужской, русский) | Вопросы заранее, пояснения — на лету с кэшем |
| База данных | Supabase | Текст учебника + кэш аудио + прогресс |
| Деплой | GitHub → Vercel | |

## Flow ученика

1. Открывает сайт → главная с картой Древнего мира
2. Выбирает раздел → список параграфов (все 31 доступны, можно повторить любой)
3. Видит иллюстрацию эпохи + краткое вступление (озвучено ElevenLabs)
4. Появляется вопрос №1 + тематическая картинка + всплывающий факт "Ты знал?"
5. Ученик отвечает голосом или текстом
6. Gemini анализирует ответ → пояснение 5-10 предложений + интересный факт + сравнение с современностью
7. Пояснение озвучивается (ElevenLabs) + яркая ассоциация для запоминания
8. Повтор для 3-5 вопросов по параграфу
9. Финал: тест по датам параграфа (интерактивные карточки)
10. Результат + медаль за параграф + кнопка "Поделиться"
11. Обобщающий тест по разделу (даты, события, персоналии из всех параграфов раздела)
12. Режим повторения — только вопросы, где ученик ошибся

## Решения по ключевым проблемам

### 1. Совместимость браузеров (голосовой ввод)
**Решение: голос + текстовый fallback**
- Web Speech API для Chrome/Edge (голосовая кнопка)
- Текстовое поле всегда доступно как альтернатива
- Автоопределение: если браузер не поддерживает Speech API → текстовый режим по умолчанию
- Работает на iPhone, Android, любом браузере

### 2. Стоимость ElevenLabs
**Решение: pre-generate + кэширование**
- Вопросы к параграфам фиксированные → озвучиваются заранее, хранятся как аудиофайлы
- Вступления к параграфам → тоже заранее
- Пояснения к ответам → генерируются на лету, но кэшируются в Supabase Storage
- Если похожее пояснение уже было → отдаётся из кэша
- Экономия до 70% вызовов API

### 3. Сохранение прогресса без авторизации
**Решение: никнейм + Supabase**
- При первом входе ученик вводит любой никнейм (без пароля)
- Прогресс сохраняется в Supabase по никнейму
- Дополнительно дублируется в LocalStorage (работает и оффлайн)
- Можно продолжить с другого устройства, введя тот же никнейм

### 4. Распознавание речи ребёнка
**Решение: умный промпт для Gemini**
- Системный промпт: "Ученик 5 класса отвечает на вопрос по истории. Его речь может содержать ошибки в произношении терминов (Пелопоннес, Фермопилы и т.д.). Интерпретируй ответ максимально лояльно, ищи суть, а не точные формулировки."
- Gemini по контексту параграфа поймёт даже искажённые термины
- Никогда не говорит "неправильно" — всегда "ты на верном пути, давай уточним..."

### 5. Mobile-first дизайн
- Крупные кнопки (минимум 48px)
- Читаемый шрифт (16px+)
- Картинки адаптивные
- Большая кнопка микрофона по центру
- Минимум скролла — один вопрос на экран

## Принятые решения

- **LLM:** Google AI Studio (Gemini) — основной
- **Картинки:** Pre-generated через Google AI Studio, статические
- **STT:** Web Speech API + текстовый fallback (работает везде)
- **TTS:** ElevenLabs, мужской голос, русский. Вопросы заранее, пояснения на лету + кэш
- **База:** Supabase (текст учебника + кэш аудио + прогресс учеников)
- **Авторизация:** никнейм без пароля + LocalStorage
- **Параграфы:** все 31 доступны сразу
- **Тесты:** по параграфу (после вопросов) + по разделу (обобщающий)
- **Дизайн:** mobile-first, один вопрос на экран
- **Тон AI:** дружелюбный, поддерживающий, никогда не критикует

## Безопасность и UX

- **Безопасность контента:** системный промпт с жёсткими ограничениями — только образовательный контент, простой язык для 5 класса, без насилия/политики
- **Система подсказок:** кнопка "Подскажи" → AI даёт наводящий вопрос, а не ответ сразу. Три уровня: лёгкая подсказка → конкретный намёк → ответ с пояснением
- **Адаптивная сложность:** отвечает хорошо → глубже и сложнее; ошибается → проще и подробнее
- **Онбординг:** при первом визите — короткий тур из 3-4 шагов ("нажми сюда, говори сюда")
- **Звуковой дизайн:** правильный ответ → приятный звук, медаль → фанфары, фоновая атмосфера эпохи
- **Загрузка:** пока AI думает → анимация (свиток разворачивается), не пустой экран

## Итоги и повторение

- **Итоги усвоения:** после каждого параграфа — сводка: сколько правильных, какие темы усвоены, где пробелы
- **Повторное прохождение:** плохо отвеченные вопросы автоматически собираются в отдельный блок "На повторение"
- **Принудительное повторение:** ученик не может получить медаль за параграф, пока не ответит правильно на все вопросы (можно пройти повторение в любой момент)
- **Итоги по разделу:** общая сводка по всем параграфам раздела перед обобщающим тестом

## Масштабируемость

- Архитектура позволяет добавить другой PDF (география, биология) без переписывания кода — только загрузить новый учебник

## Frontend

- **Подход:** Frontend Design Skill для создания уникального визуального стиля
- **UI:** Tailwind CSS + shadcn/ui — кастомная "античная" тема
- **Анимации:** Framer Motion (свитки, карточки, медали, пульсация микрофона)
- **Карты:** react-leaflet (интерактивные карты с метками событий)
- **Стейт:** Zustand (лёгкий, без бойлерплейта)
- **Аудио:** ElevenLabs SDK + Web Audio API
- **Голос:** Web Speech API

### Визуальная тема
- Фон: тёплый песочный (#F5E6D3), текстура пергамента
- Акцент: терракота (#C75B39)
- Текст: тёмно-коричневый (#3D2B1F)
- Успех: оливковый (#6B8E23)
- Заголовки: Merriweather / Playfair Display
- Текст: Inter / Nunito (округлый, дружелюбный для детей)

### Структура страниц
```
/                       → Главная (карта Древнего мира, выбор раздела)
/section/[id]           → Раздел (список параграфов + обобщающий тест)
/paragraph/[id]         → Параграф (вопросы, карта, иллюстрации)
/paragraph/[id]/quiz    → Тест по датам параграфа
/paragraph/[id]/review  → Повторение ошибок
/section/[id]/quiz      → Обобщающий тест по разделу
/section/[id]/summary   → Итоги по разделу
```
